# 👨‍🍳 員工端 UI 畫面設計

## 🎯 設計目標

員工端界面專為餐廳工作人員設計，重點在於：
- **效率優先**: 快速處理訂單和更新狀態
- **清晰明瞭**: 重要資訊一目了然
- **觸控友好**: 適合忙碌環境的快速操作
- **即時更新**: 訂單狀態即時同步

## 🔧 API 整合說明

### 主要 API 端點
```javascript
// 獲取待處理訂單
GET /api/orders/pending

// 獲取訂單詳情  
GET /api/orders/{orderId}

// 更新訂單狀態
PUT /api/orders/{orderId}/status
{
  "status": "PREPARING"
}

// 獲取員工統計
GET /api/orders/staff/{staffId}/stats

// 實時通知 (WebSocket)
ws://localhost:8080/notifications
```

### 狀態同步機制
```javascript
// 訂單狀態監聽
const orderStatusWS = new WebSocket('ws://server/orders/status');
orderStatusWS.onmessage = (event) => {
  const update = JSON.parse(event.data);
  updateOrderUI(update.orderId, update.status);
};

// 新訂單通知
const newOrderWS = new WebSocket('ws://server/orders/new');
newOrderWS.onmessage = (event) => {
  const newOrder = JSON.parse(event.data);
  showNewOrderNotification(newOrder);
};
```

# 🍽️ 顧客端 UI 畫面設計

## 📱 畫面結構總覽

### 主要畫面流程
```
啟動頁 → 登入註冊 → 主頁 → 菜單瀏覽 → 菜品詳情 → 購物車 → 訂單確認 → 付款 → 訂單追蹤
```

---

- **Logo**: Ranbow 品牌標識

### 購物車功能

#### 商品管理
- **數量調整**: [─] 和 [+] 按鈕調整數量
- **刪除商品**: 左滑刪除或垃圾桶圖標
- **編輯特殊要求**: 點擊可重新編輯
- **清空購物車**: 一鍵清空所有商品

#### 費用計算
- **小計**: 所有商品價格總和
- **稅金**: 自動計算 (10%)
- **總計**: 小計 + 稅金
- **即時更新**: 數量變更時即時重新計算

#### 操作按鈕
- **繼續購物**: 返回菜單頁面
- **去結帳**: 進入訂單確認頁面
